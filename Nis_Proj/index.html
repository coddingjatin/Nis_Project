College Registration Process

Step 1: Registration Through ABC Website

Visit on www.abc.gov.in

Click on My Account -> Select "Student"

You will directed to the DigiLocker - signin page

Step 2: DigiLocker Login

Sign in to your account using your registered mobile number

Enter the OTP recieved on your mobile number

Then Proceed

Step 3: Personal Information

Click on Select Institution Type -> Select "University"

Select Institution Name -> Select "COEP Technological University"

Step 4: ABC Registration Completed

Note Down your ABC ID

This ABC ID will Be Used For Futher Processes

Step 5: Fill Anti-Ragging Form

Enter Your All the personal details

In University details Enter University code as U-1257

In University details Enter University Name : COEP Technological University

Fill All other Information

Submit the Form

×

Payment Procedure

1.Go to www.Sbicollect.com -> Select Educational Institutes

2. Search for COEP Technological University

3. Select payment category FY 24-25 University Fees

4. Fill The Personal Information

5. Select The Payment Option And Your Complete the payment
